Detailed Description

Manages creation of server statistics, which are used for diagnostics.

Server statistics are gathered in two ways. First, this object maintains a stats structure that modules can fill in by acquiring a Lock and modifying directly. This stats structure is copied every time stats are requested. Second, when stats are requested, getCurrent() will ask certain modules (such as RaftConsensus) to fill in the current information into a stats structure.
